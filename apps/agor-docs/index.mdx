The multiplayer-ready, spatial layer that connects Claude Code, Codex, Gemini, and any agentic coding tool into one unified workspace.

<Note>
  **TL;DR:** Agor is a multiplayer spatial canvas where you coordinate multiple AI coding assistants
  on parallel tasks, with GitHub-linked worktrees, automated workflow zones, and isolated test
  environments—all running simultaneously.
</Note>

## Quickstart

```bash
npm install -g agor-live
agor init
agor daemon start
agor open
```

**Try in GitHub Codespaces (up in 1–2 minutes):**

[![Open in GitHub Codespaces](https://github.com/codespaces/badge.svg)](https://github.com/codespaces/new?skip_quickstart=true&machine=basicLinux32gb&repo=1069864589&ref=main&devcontainer_path=.devcontainer%2Fdevcontainer.json&geo=UsWest)

See the [Getting Started Guide](/guide/getting-started) for detailed instructions, or the [Development Guide](/guide/development) for Docker setup.

## Key Features

- **Agent swarm control** across Claude Code, Codex, and Gemini
- **Multiplayer spatial canvas** for parallel work
- **Session trees** with fork and spawn workflows
- **Zone triggers** for workflow automation
- **Isolated environments** per worktree
