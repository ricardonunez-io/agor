# Getting Started

Get Agor running in under 2 minutes.

## Installation

```bash
npm install -g agor-live
```

## Quick Start

```bash
# 1. Initialize (creates ~/.agor/ and database)
agor init

# 2. Start the daemon
agor daemon start

# 3. Open the UI
agor open
```

**That's it!** The UI will guide you through onboarding, adding repositories, and creating your first session.

## Troubleshooting

### Port 3030 already in use

```bash
# Find and kill the process
lsof -ti:3030 | xargs kill -9

# Or configure a different port
agor config set daemon.port 4030
agor daemon start
```

### Daemon not starting

```bash
# Check daemon status
agor daemon status

# View logs
agor daemon logs

# Check health endpoint
curl http://localhost:3030/health
```

### Can't connect to daemon

Make sure the daemon is running:

```bash
agor daemon status
# If not running:
agor daemon start
```
