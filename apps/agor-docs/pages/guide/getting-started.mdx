# Getting Started

Quick start guide for Agor.

## Installation

```bash
git clone https://github.com/mistercrunch/agor
cd agor
docker compose up
```

## First Steps

1. Open http://localhost:5173
2. Login with `admin@agor.live` / `admin`
3. Create your first session
4. Start orchestrating!
